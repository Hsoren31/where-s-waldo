# Where's Waldo

## About the Project

The goal of this project is to build a platform where visitors can play _Where's Waldo_ against the clock. It serves as part of the Odin Project curriculum, bringing together everything I've learned while introducing new challenges.

## Built with

- React
- NodeJS
- Express
- Prisma ORM
