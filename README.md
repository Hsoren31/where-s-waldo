## About the Project

This project is an interactive photo tagging game inspired by Where’s Waldo, designed to bring together both frontend and backend development skills. Users are presented with a detailed image containing multiple hidden characters, and their goal is to locate each one as quickly as possible.

When a user clicks on the image, a targeting box appears at the selected location, allowing them to choose which character they believe they’ve found. The application then communicates with the backend to validate whether the selected character is correctly positioned within that area. If correct, a marker is placed on the image; if not, the user receives feedback and can try again.

The game tracks the total time taken to find all characters, ensuring score integrity by handling timing on the server side. Once all characters are identified, users can submit their name to be recorded on a leaderboard, adding a competitive element to the experience.

It is part of the [Odin Project](https://www.theodinproject.com) curriculum.
